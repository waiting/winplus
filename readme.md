Windows实用C++工具库 - WinPlus
============================

WinPlus是一个轻量级的C++库，提供一些windows程序开发中的常用功能。

最低编译器：VS2010
相关模块:

	compoundfile - 复合文件相关
	console      - 控制台相关
	definitions  - 基本类型定义，系统头文件包含
	graphics     - 绘图相关支持，一些结构和绘图计算例程，内存DC，内存图片
	ini          - INI配置文件支持
	md5          - MD5加密支持
	picture      - 图片文件相关
	resource     - 资源相关
	shell        - 外壳相关
	strings      - 字符串相关
	system       - 系统相关
	time         - 时间相关
	winctrl      - 窗口控件相关
	winplus      - 主头文件，包含winplus全部头文件
	wordslib     - 词库相关
